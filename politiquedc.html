Politique de Confidentialité

Dernière mise à jour : [05/01/2025]

Introduction

Bienvenue sur LyCloud Agency. Nous attachons une grande importance à la protection de vos données personnelles et nous nous engageons à respecter votre vie privée. Cette Politique de Confidentialité explique quelles données nous collectons, comment nous les utilisons et quels sont vos droits.

Responsable du traitement des données

L’entreprise LyCloud Agency, immatriculée sous le numéro SIRET 94024893300019 , est responsable du traitement des données personnelles collectées sur ce site.

Données collectées

Nous collectons uniquement les données nécessaires à la gestion de notre activité et à la fourniture de nos services. Les données collectées incluent :

Données de compte : nom, prénom, adresse e-mail
Données de paiement : via notre prestataire Stripe
Données de communication : messages envoyés via notre support
Données techniques : adresse IP

Finalité de la collecte des données

Vos données sont collectées afin de :

Gérer vos commandes et paiements (via Stripe) (Exécution du contrat)
Assurer le support client et répondre à vos demandes (Exécution du contrat)
Améliorer nos services et notre site web (Intérêt légitime)
Respecter nos obligations légales et comptables (Obligation légale)

Partage des données

Nous ne partageons pas vos données avec des tiers, sauf dans les cas suivants :
Stripe : pour la gestion des paiements sécurisés
Obligations légales : en cas de demande des autorités compétentes

Transferts hors Union Européenne : Certaines données (ex. : paiements via Stripe) peuvent être transférées hors de l’UE. Nous nous assurons que ces transferts respectent les normes de protection des données exigées par le RGPD.

Sécurité des données

Nous mettons en place des mesures de sécurité techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, perte ou vol.

Durée de conservation

Les données de facturation sont conservées pendant 10 ans (obligation légale).
Les autres données (compte, communication, logs) sont conservées aussi longtemps que nécessaire pour fournir nos services.

Vos droits

Conformément au RGPD, vous avez les droits suivants :

Droit d’accès : demander quelles données nous avons sur vous
Droit de rectification : modifier vos informations si elles sont incorrectes
Droit à l’effacement : demander la suppression de vos données (sauf obligations légales)
Droit à la portabilité : récupérer vos données dans un format lisible
Droit d’opposition : refuser l’utilisation de vos données à certaines fins

Pour exercer ces droits, contactez-nous.

Contact : Si vous avez des questions sur cette Politique de Confidentialité, vous pouvez nous contacter à : contact.lycloud@gmail.com
